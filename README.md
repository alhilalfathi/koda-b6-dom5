# Navbar Responsive
Navigasi bar yang dapat menyesuaikan dengan layar device yang mengaksesnya

# Hasil
## 1. Jika ukuran layar <768 atau dibawah medium maka tampilan navbar akan mendatar
![<768](readme-img/1.png
)
## 2. Jika ukuran layar lebih dari 767 dan kurang dari 1025 maka tampilan navbar akan menjadi menurun
![>767 <1025](readme-img/2.png)
## 3. Jika ukuran layar melebihi 1024 maka navbar akan kembali ke style default
![>1024](readme-img/3.png)
